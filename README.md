## 简介

本项目是提供B站web移动端api接口项目

## 加入项目

如果想加入本项目开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可

## 运行

启动项目

```
yarn install
```

```
yarn start
```

接口文档

```
yarn docs
```
